Steps to sign git with visual studio code 

gpg --list --keys

gpg --full generate keys

done all the steps and copy secretid 

gpg --armor --export secretid

paste the given content in gpg key in github profile

git config --global user.signingkey secretid

code .gitconfig

add this lines in above file
    [user]
        signingkey = secretid
    
git true command

code .gitconfig

add 
    [commit]
        gpgsign = true
save file
